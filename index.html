1
Clone the Repository
Download Browser Harness to your machine.
# Clone the repo
git clone https://github.com/marceli1404/browser-harness.git
cd browser-harness
2
Install Dependencies
Install Node.js packages and Playwright browsers.
# Install npm packages
npm install

# Install browser engines (Chromium, Firefox, WebKit)
npx playwright install
3
Test Installation
Verify everything is working correctly.
# Test with Chromium
node browser.js open https://example.com

# Test with Firefox
node browser.js open https://example.com firefox

# Test mouse movement
node browser.js move https://example.com 100 100 500 400
4
AI Integration (Optional)
Give your AI agent browser access.
# Add to your AI's system prompt:
"You have access to a browser automation tool.
Use: node /path/to/browser.js open <url>"

# Or set up MCP server for Claude Desktop
See README.md for full instructions
5
Start Automating
You're ready! Try these commands:
# Take a screenshot
node browser.js screenshot https://google.com

# Fill a form
node browser.js fill https://google.com '[name=q]' 'search'

# Extract page text
node browser.js text https://news.ycombinator.com
